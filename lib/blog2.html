CSS Variables

Syntax of the var() Function The var() function is used to insert the value of a CSS variable. The syntax of the var() function is as follows: var(--name, value) name: Required. The variable name (must start with two dashes) value: Optional. The fallback value (used if the variable is not found) Note: The variable name must begin with two dashes (--) and it is case sensitive!

How var() Works

First of all: CSS variables can have a global or local scope. Global variables can be accessed/used through the entire document, while local variables can be used only inside the selector where it is declared. To create a variable with global scope, declare it inside the :root selector. The :root selector matches the document's root element.
